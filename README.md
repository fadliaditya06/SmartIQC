## 📝 Deskripsi Proyek
**Smart IQC Apps** adalah solusi transformasi digital untuk proses *Incoming Quality Control* (IQC). Aplikasi ini menggantikan sistem inspeksi konvensional yang berbasis kertas menjadi platform digital yang terintegrasi untuk meningkatkan akurasi data, mempermudah pelacakan *defect*, memantau kondisi lokasi inspeksi secara *real-time* serta memfasilitasi verifikasi *wriststrap* harian.

---

## 🚀 Fitur Utama

### 1. 📦 Smart IQC
* **Checklist:** Memberikan status oke atau *not oke* (*defect*) pada setiap *checkpoint* material secara dinamis.
* **Open Points:** Halaman khusus untuk melihat seluruh riwayat data yang terdeteksi *defect* (*not oke*) dan untuk memperbaiki temuan *defect*, merubah PIC, serta memverifikasi hasil perbaikan untuk menutup status temuan.
* **All Points:** Melihat seluruh riwayat riwayat inspeksi untuk semua status baik oke maupun *not oke* (*defect*).

### 2. 🌡️ Temperature & Humidity
* **Dashboard Monitoring:** Melihat grafik nilai *temperature* dan *humidity* secara historis dari waktu ke waktu.
* **Temperature Humidity Status:** Memantau status kondisi *temperature* dan *humidity* berdasarkan lokasi.
* **Temperature Humidity Check:** Fitur input nilai *temperature* dan *humidity* secara harian pada setiap lokasi.

### 3. 🛡️ Wriststrap Check
* **Daily Wriststrap Check:** Melihat status *wriststrap* serta dapat melakukan pengecekan *wriststrap* untuk menentukan apakah *wriststrap* dalam kondisi OK atau *Not Good* (NG) secara harian pada setiap shift.
